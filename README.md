# grsim_ros_bridge
grSim(robocup soccer small size league simulator) bridge for ROS(robot operating system)

grSim can download from this repository(https://github.com/RoboCup-SSL/grSim)

# how to install  
1.build and install grSim  
2.cd catkin_ws/src  
3.git clone https://github.com/hakuturu583/grsim_ros_bridge  
4.cd ../  
5.catkin_make  

# how to run  
roslaunch grsim_ros_bridge robocup_ssl.launch
